Using ssh keys to work with github

Today i'll be documenting the way I set up my ssh keys to work with github repositories

Generate ssh keys

ssh-genkey -t TYPE -C "EMAIL@EMAIL.COM"
See what do you want to do as for the type, passphrase or not ...

Configuration

create .ssh/config file then configure it to match your hosts and the connection types.
See for details on configurations.

Add ssh key to your ssh-agent

Start the ssh-agent; eval "$(ssh-agent -s)"
Add your key to it; ssh-add ~/.ssh/KEY_NAME.

Add your key to github

Copy your public key to the clipboard; "xclip -sel clip - ~/.ssh/KEY_NAME.pub "
Got to Settings - SSH & GPG KEYS - add ssh key - pubste your key -- OK

Testing

Test your connection with; ssh -vT git@github.com

Case your remote url is stil in https

git remote set-url origin git@github.com:username/your-repository.git
