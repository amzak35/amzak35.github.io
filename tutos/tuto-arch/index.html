Setting up a Gnu/Linux operating system for the Desktop/Laptob usage



Goals

The aim from this tutorial is to learn to install and setup Archlinux.

Why Arch? Well, Other than it being my daily driver therefore I’m more familiar and confortable setting it up, I chose Arch because it gives you full control on the programs and utilities you can install, witch makes it easy to showcase and learn a lot of things about the way Gnu/Linux operates.

Another goal is to showcase and introduce you to the programs I run on my computer that could replace a lot of proprietary garbage a lot of us got used to.

Installing ArhLinux

In this first tutorial I’m going to set up Arch Linux from the ground up (not as in LFS though) to get to a fully functioning system with a minimal gui, hopefully my own window manager (project in the works) … Stay tuned folks!
