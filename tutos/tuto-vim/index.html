What is Vi

Vi is a terminal based text editor, a very powerful one, with a somewhat steep learning curve. Lot of people (Even some technical savvy ones) are afraid of it, they find it complicated to use and non user friendly, when in reality, all it takes is the willingness to try it without the prejudice surrounding it.

I mean, if you just take the time to understand it’s logic and mechanics you’ll be amazed, and you’ll find the “classical” that other text editors work is non efficient.
